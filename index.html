About these tools

These tools allow you to run and debug your Human Resource Machine programs (you can copy and paste them between this website and the game) and to create labels or comments to use in the game by typing in some text.

The code used by these tools is MIT-licensed and available for the browser on Github and NPM.

The component modules behind these tools are also MIT-licensed and available for node.js on Github and NPM:

  • hrm-cpu Github NPM
  • hrm-level-data Github NPM
  • hrm-draw Github NPM
  • hrm-parser Github NPM
  • hrm-image-encoder Github NPM
  • hrm-image-decoder Github NPM

About Human Resource Machine

Human Resource Machine is a puzzle game. In each level, your boss gives you a job. Automate it by programming your little office worker! If you succeed, you'll be promoted up to the next level for another year of work in the vast office building. Congratulations!
